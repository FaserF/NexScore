# FactQuest - User Guide 🧠

FactQuest is a trivia game designed for car rides or group play, focusing on interesting facts and bizarre stories.

## 🎴 Game Categories

- **Random Facts**: Curated verified facts from various fields (Science, History, Nature).
- **Dumb Ways to Die**: Real-life bizarre fatal accidents, told in a subtle "Black Stories" style that keeps players guessing before revealing the full explanation.

## 🕹 Features

### 1. Sources & Verification
Every card includes a **Source** button that opens a verified URL (Wikipedia, news articles, scientific journals) to allow players to read more or verify the fact.

### 2. Category Filtering
You can toggle categories at the start of the game or **during gameplay** via the settings icon in the top right.
- Want only fun facts? Toggle off "Dumb Ways to Die".
- Want pure "Black Stories" style? Toggle off "Random Facts".

### 3. Undo Button
Accidentally tapped "Next Card"? Use the **Undo** button in the AppBar to revert to the previous fact.

### 4. Multiplayer
FactQuest supports multiplayer synchronization. When a host draws a card, it appears on all connected clients' screens simultaneously.
