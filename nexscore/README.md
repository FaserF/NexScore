# NexScore 🏆

NexScore is a premium, cross-platform score tracking application designed for serious board and card game enthusiasts. Built with **Flutter 3.41.4**, it offers a seamless experience across **Web**, **iOS**, and **Android**.

![NexScore Banner](https://placehold.co/1200x400/2c3e50/white?text=NexScore+Multi-Game+Score+Tracker)

## 🎯 Supported Game Modules

1.  **Wizard** (Standard, Lenient, Extreme variants)
2.  **Qwixx** (Digital scoresheet with automated scoring)
3.  **Schafkopf** (Solo, Sauspiel, Wenz with Bavarian rules)
4.  **Kniffel / Yahtzee** (Full scoresheet with bonus logic)
5.  **Phase 10** (Original, Masters, and Duel variants)
6.  **Darts X01** (301, 501, 701, 1001 with bust detection)
7.  **Rommé** (Multi-round penalty tracking)
8.  **Arschloch / President** (Rank tracking, card exchange rules)
9.   **SipDeck** (Multiple cards, party-ready, 18+ warning)
10.  **FactQuest** (Trivia for car rides with 360+ facts and "Black Stories" style bizarre deaths)
11.  **Volleyball** (Official Indoor & Beach scoreboard with PDF match reports and digital signatures)
12.  **WayQuest** (Interactive travel companion with location-based triggers)

... and more!

## ✨ Premium Features

- **🌐 Web-First UX**: Robust "Enter" key support and physical keyboard shortcuts.
- **🌍 Full Internationalization**: 100% localized in **English** and **German**.
- **🌗 Smart Theming**: Integrated Light and Dark modes with specialized FlexColorScheme.
- **💾 Offline-First**: Reliable local storage with optional Firestore cloud synchronization.
- **📊 Detailed Statistics**: Win-rate analytics and comprehensive match history.
- **📸 Share-To-Story**: Capture beautiful game result scorecards and challenges to share on Instagram, WhatsApp, and more.
- **🔊 Living Audio**: Integrated Sound Effects (SFX) and Text-To-Speech (TTS) for an immersive game night.
- **🚀 Agile Releases**: Automated CI/CD pipelines and Docker production support.
- **🦟 Debugging Tools**: Integrated global system logging and log export for easy troubleshooting.
