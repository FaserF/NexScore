# Changelog

## [0.3.0] - 2026-03-07
### Added
- **Arschloch**: Officially supports 2-player mode (with a recommended warning banner for authentic experience).
- **SipDeck**: Massive expansion! Added 110+ premium new cards localized in English and German.
- **BuzzTap**: Massive expansion! Added 120+ premium new scenarios and tasks localized in English and German.
- **Smart Filtering**: New 2-Player mode optimization natively filters out group-centric cards dynamically in SipDeck and BuzzTap.
- **Drink Intensity Settings**: Customize drinking challenges with Chill (0.5x), Normal (1x), Extreme (2x), and Custom modes.
- **Swipe Gestures**: SipDeck and BuzzTap cards now support Tinder-style horizontal swipe navigation with smooth animated transitions.
- **Testing**: Added rigorous programmatic unit tests for 2-player content validation and game settings calculations.

### Fixed
- **SipDeck**: Fixed mobile interface overflow where users could not scroll to the Start button.
- **Arschloch**: Properly registered missing navigation router path to prevent missing screen errors.
