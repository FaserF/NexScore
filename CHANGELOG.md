# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-03-07

### Added
- **SipDeck Task Filtering**: Independent toggles for Dares, Social Interaction, Messaging, and Physical tasks.
- **Generic Scoreboard**: A flexible, round-based table for tracking scores in any game.
- **Improved PWA Performance**: Optimized touch handling and CSS for 120Hz scrolling on Android devices.
- **Enhanced Documentation**: Comprehensive guides for all games and features in the `/docs` directory.
- **New Icons & UI**: Updated games list with better category tags and glassmorphism refinements.

### Changed
- Rebranded remaining "ScorePlay" references to **NexScore**.
- Optimized 2-player mode for party games.
- Updated localizations for English and German.

### Fixed
- Fixed PWA "laggy" scrolling feel on high-refresh rate displays.
- Resolved various lint errors and dependency warnings.

## [0.1.0] - 2026-03-04
- Initial release under the NexScore name.
- Support for Wizard, Qwixx, Schafkopf, Kniffel, Phase 10, Darts, Rommé, Arschloch, SipDeck, BuzzTap, and WayQuest.
- Firebase integration for cloud sync.
- Multi-language support (EN/DE).
