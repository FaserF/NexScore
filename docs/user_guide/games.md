# Supported Games 🎮

NexScore provides specialized scoreboards for several popular games to make tracking as easy as possible.

## 🃏 Wizard

The Wizard scoreboard handles all the complex point calculations for you.

- **Bid Entry**: Enter how many tricks each player expects to win.
- **Result Entry**: Enter how many tricks they actually won.
- **Automatic Calculation**:
    - Correct bid: +20 points + 10 per trick.
    - Incorrect bid: -10 points per trick difference.

## 🎲 Qwixx

Track your Qwixx game progress without needing a pen and paper.

- **Row Tracking**: Tap numbers in the Red, Yellow, Green, and Blue rows.
- **Locks**: Lock rows when someone reaches five crosses and the last number.
- **Misthrows**: Track up to 4 misthrows (-5 points each).
- **Auto-Score**: Points are calculated based on the number of crosses in each row using the triangular number sequence (1, 3, 6, 10, ...).

## 📊 Generic Scoreboard

For any other game, use the Generic scoreboard.

- **Round-based**: Add points after each round.
- **Quick Entry**: Just enter the final scores if you only want to record the winner.
- **Flexibility**: Works for everything from Uno to Monopoly.
