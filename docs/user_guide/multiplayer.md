# Multiplayer & Remote Play 🌐

NexScore features a powerful, real-time multiplayer system that allows you to track scores with friends, whether they are in the same room or on the other side of the world.

## 🚀 Getting Started

Multiplayer is available for all supported games. To start a multiplayer session:

1.  **Open a Game**: Select any game from the Home screen.
2.  **Tap the Multiplayer Icon**: Look for the "people" or "globe" icon in the top right corner of the setup screen.
3.  **Choose Host or Join**:
    *   **Host**: Create a new session. You will receive a unique **6-digit Game Code**.
    *   **Join**: Enter a Game Code provided by a friend to jump into their session.

## 🏠 Hosting a Game

When you host a game, your device becomes the "source of truth" for the session.

-   **Share the Code**: Give the 6-digit code to your friends.
-   **Lobby Management**: You can see players joining in real-time.
-   **Start Game**: Once everyone is in, start the game. All connected devices will transition to the game screen automatically.

## 🛰️ Joining a Game

Joining is simple and doesn't require an account.

-   **Enter Code**: Type in the code provided by the host.
-   **Syncing**: Your device will automatically pull the current player list and game settings.
-   **Follow Along**: As the host or other players update scores, your screen will update instantly.

## 🎮 Remote Play Features

NexScore is optimized for remote play (e.g., over Discord or Zoom):

-   **Global Sync**: Any change made by any player (if permitted by the game type) is reflected for everyone.
-   **Low Latency**: Built on a high-performance socket/cloud overlay to ensure minimal delay.
-   **Cross-Platform**: Play seamlessly between Android, iOS, and Web (PWA).

## 💡 Pro Tips

-   **Stable Connection**: Ensure you have a reliable internet connection for the best experience.
-   **Privacy**: Only share your game code with people you want to play with.
-   **Battery**: Multiplayer real-time sync can use more battery than local play; keep a charger handy for long sessions!

---

*Note: Multiplayer requires an active internet connection. Offline play is always available for local sessions.*
